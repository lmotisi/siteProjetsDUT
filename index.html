Le Projet

Plants Vs. Zombies est un jeu de tower defense développé par PoCap Games et sorti en 2009. Dans le cadre de notre projet de fin de 1ère année de DUT Informatique, nous avons réalisé en binôme une version JAVA du jeu dans lequel le jardin est constitué de cases sur lesquelles il est possible de planter différentes plantes, le but étant de survivre un certain nombre de tours aux attaques des zombies.
